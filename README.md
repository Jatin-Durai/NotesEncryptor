## Description

This Node.js application provides a secure environment for user registration and login. It incorporates RSA encryption for secure data transmission and SHA for password hashing. The application is built with Express and uses MongoDB for data storage.

## Features

- User Registration
- Secure User Login
- RSA Encryption
- SHA Password Hashing
- Express Middleware for Authentication

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js v18.16.0 installed
- MongoDB installed and running
- npm package manager

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Jatin-Durai/secure-node-app.git
   ```

2. Navigate to the project directory:

   ```bash
   cd secure-node-app
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

1. Start the application:

   ```bash
   npm start
   ```

2. Open your browser and go to [http://localhost:3000](http://localhost:3000).

## Configuration

- Modify the MongoDB connection string in `config/database.js` to match your MongoDB setup.
- Adjust other configuration parameters as needed in the respective files.
